# 🚀 Missão UC2: Tecnologia em Desenvolvimento de Sistemas (TDS)

Bem-vindo ao repositório central da **Unidade Curricular 2 (TDS)**! 🌌 
Este espaço é o diário de bordo de uma jornada pelo universo do desenvolvimento frontend e lógica de programação. Aqui, o código puro encontra a interatividade, transformando conceitos abstratos em aplicações reais.

## 🛸 O Que Tem Neste Universo?

Este repositório está dividido em diferentes galáxias de conhecimento, cada uma focada em uma habilidade essencial de engenharia de software:

*   🐛 **BugsLife**: Uma constelação de 16 mini-projetos práticos e cotidianos. Inclui lógicas para cinemas, revendas, farmácias, cálculo de IMC, tabuadas e até um simulador de caixa eletrônico.
*   🌳 **Manipulando o DOM**: Onde o código ganha vida. Exercícios focados em capturar eventos e alterar a interface dinamicamente (como supermercados, lanhouses e calculadoras de veículos)[cite: 1].
*   🔀 **Condicionais & Switch Case**: O centro de tomada de decisões. Aplicações que lidam com fluxos complexos, como cálculo de peso ideal, fuso horário e raízes quadradas[cite: 1].
*   📦 **Programas Node**: A sala de máquinas. Scripts rodando diretamente no terminal, abordando desde lógicas de veterinária até soma de números e cálculos de quadriênios[cite: 1].
*   🧩 **Exercícios de Lógica**: Baterias de testes (Exercício A ao H) para refinar a capacidade de resolução de problemas algorítmicos[cite: 1].
